## Основная информация

**Задача**: С помощью машинного обучения предсказать количество заказов такси в аэропортах.
**Цель**: привлечение дополнительного количества водителей во время пиковой загрузки.

## Описание датасета

Используются исторические данные о заказах такси в аэропортах.

**Обучающий признак**
datetime - дата создания заказа

**Целевой признак**
num_orders - количество заказов

## Используемые библиотеки python
- Matplotlib
- NumPy
- Pandas
- Scikit-learn
- statsmodels


## Дополнительная информация

Значение метрики RMSE на тестовой выборке должно быть не больше 48.
